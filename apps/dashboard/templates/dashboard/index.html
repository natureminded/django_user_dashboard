User Dashboard

an MVC architectured application which allows `CRUD` events for Users, Messages and Comments.

Manage

In this application, Admins are able to add, remove and edit other users.

Message

Users can leave messages to other users, and respond to one another.

Edit

Admins can edit other user's information. Normal users can edit their own profile.
